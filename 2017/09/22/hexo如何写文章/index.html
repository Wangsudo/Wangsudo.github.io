设置文章属性

新建的文章中会出现以下代码，默认的属性设置

在两条---之内可以设置文章的属性

属性
Setting Description Default
layout Layout post或page
title 标题文章
date 创建文章的日期 文件的创建日期
update 修改日期 文件的修改日期
comments 是否开启评论 true
tags 文章标签 默认为空自己设置
categories 文章分类 默认为空自己设置
permalink url中的名字 文件名



每个hexo的主题下，都有对文章特殊的属性设置，例如文章背景图片呀等等


ps:

属性名后面的冒号应是英文冒号;

且冒号后要加空格之后再写具体内容。
